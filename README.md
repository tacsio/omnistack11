#  OmniStack 11 by [@Rocketseat][rocketseat] ![rocketlogo][rocketlogo]

```
$1 docker run -p 27017:27017 -it mongo
$2 yarn install
$2 yarn dev
```

```
yarn install
yarn start
```

```
yarn install
yarn start
# Run on expo
```
