#  OmniStack 11 by [@Rocketseat][rocketseat] ![rocketlogo][rocketlogo]

![applogo][applogo]

> Backend

```
$1 npx knex migrate:latest


$1 docker run -p 27017:27017 -it mongo
$2 yarn install
$2 yarn dev
```

> Frontend

```
yarn install
yarn start
```

> Mobile

```
yarn install
yarn start
# Run on expo
```
